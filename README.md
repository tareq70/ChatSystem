# 💬 ChatSystem — Real-Time Chat Application

A modern real-time chat application built with **ASP.NET Core (Razor Pages)** and **SignalR**, supporting private messaging, group chats, file sharing, and live updates.

---

## 🚀 Overview

**ChatSystem** is an open-source chat platform designed to provide a smooth and responsive real-time communication experience. It leverages SignalR for instant updates and Razor Pages for a clean, server-rendered UI.

---

## ✨ Features

- 🔒 **Private Chats & Group Conversations**
  - One-to-one messaging
  - Group creation & management
  - Admin roles & member control

- 📎 **File & Image Sharing**
  - Upload and download files
  - Image preview support

- ⚡ **Real-Time Messaging**
  - Instant updates using SignalR
  - Seamless live communication

- ⌨️ **Typing Indicators**
  - Know when someone is typing

- ✅ **Read Receipts**
  - Track message delivery and reading status

- 🎨 **Custom UI**
  - Built with Razor Pages + Bootstrap
  - Clean and responsive design

- 🚀 **Optimistic UI**
  - Messages appear instantly before server confirmation

---

## 🛠️ Tech Stack

- **.NET 10 / C#**
- **ASP.NET Core (Razor Pages)**
- **SignalR** (Real-Time Communication)
- **Bootstrap** (UI Styling)
- **Layered Architecture** (Services, DTOs, etc.)
